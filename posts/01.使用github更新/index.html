1.测试github功能

提交你的修改至本地库。

查看当前修改状态。

git status

添加所有修改过的文件。你也可以只添加某个文件。

git add .

“Add a new post” 是 commit message.

git commit -m "更新"

将你的修改推至远程库

在 lilian-blog/public 目录下，将修改推至远程库。

git push -u origin master
